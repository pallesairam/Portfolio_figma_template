# Task_Figma

Live demo: https://delightful-daifuku-e5194e.netlify.app/

## About
Task_Figma is a responsive React + Vite project converted from a Figma design. It showcases a modern portfolio/landing UI with smooth interactions and a mobile-friendly navigation.

## Features
- Responsive layout for desktop and mobile
- Mobile navigation drawer with smooth transitions
- Resume  button and section anchors
- Built with Vite for fast dev server and optimized builds
- Tailwind CSS  for styling

## Tech stack
- React
- Vite
- Tailwind CSS 
- react-icons (used for menu / close icons)
- Netlify (deployed)
